This project is a console-based Pacman game developed in Java using basic programming concepts. It focuses on implementing the core game logic (backend) without any graphical user interface (GUI).

The game simulates Pacman moving inside a maze, collecting dots, and avoiding ghosts.

🎯 Features
2D maze using char[][]
Pacman movement
Dot collection system
Ghost movement (random or chasing)
Collision detection
Win and lose conditions
Console-based rendering
Step limit to prevent infinite execution
